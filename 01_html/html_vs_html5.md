## What is HTML?

HTML (HyperText Markup Language) is used to create the structure of web pages.

It is a markup language, not a programming language.
It only structures content like:

- Headings
- Paragraphs
- Images
- Links
- Forms
- Tables

Older versions like HTML 4.01 had limited features.


---

## What is HTML5?

HTML5 is the latest and improved version of HTML.

It introduced modern features that make websites more powerful and interactive.

HTML5 supports:

- Semantic tags
- Audio and Video (without plugins)
- Canvas for graphics
- Local Storage
- Better form inputs
- Mobile-friendly design


---

## Key Differences Between HTML and HTML5

1. Doctype

Old HTML:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">

HTML5:
<!DOCTYPE html>

HTML5 has a much simpler doctype.

---

2. Semantic Tags

Old HTML used only div and span.

HTML5 introduced meaningful tags like:
header
section
article
footer
nav

These improve readability, SEO and accessibility.

---

3. Multimedia Support

Old HTML required Flash plugin for audio and video.

HTML5 provides built-in support:

audio tag
video tag

No external plugin needed.

---

4. Storage

Old HTML only supported cookies.

HTML5 supports:
- LocalStorage
- SessionStorage

---

5. Graphics

Old HTML had no built-in graphics support.

HTML5 introduced:
- Canvas
- SVG

---

## Advantages of HTML5

- Cleaner and simpler code
- Better multimedia support
- No need for Flash
- SEO friendly
- Mobile friendly
- Supports modern web applications
- Better performance

---

## Disadvantages of HTML5

- Some features may not work in very old browsers
- Local storage can have security risks
- Advanced features require JavaScript knowledge

---

## Final Conclusion

HTML is the foundation of web development.
HTML5 is the modern, improved version of HTML.
Today, all websites use HTML5.
Learning HTML5 is essential for frontend development.
